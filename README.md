# Knott’s Seasonal Countdown Board

A live countdown dashboard tracking the major seasonal events at Knott’s Berry Farm.

**Live Site:**
https://calchan92.github.io/knotts-event-countdown/

---

##  Features

*  Real-time countdowns to major events:

  * Boysenberry Festival
  * Ghost Town Alive!
  * Knott’s Scary Farm
  * Knott’s Merry Farm

*  Dynamic seasonal effects:

  * Boysenberry Festival → drifting berry particles
  * Ghost Town Alive → desert dust
  * Scary Farm → green fog that builds throughout the year
  * Merry Farm → layered snowfall with parallax

*  Smart event states:

  * **Before event** → countdown
  * **During event** → *LIVE NOW*
  * **After event** → *Event Over*

*  Progress tracking bars for each event

---

## Built With

* HTML
* CSS (custom animations & effects)
* Vanilla JavaScript (no frameworks)

---

## How It Works

* Countdown timers update every second using JavaScript
* Seasonal effects activate automatically based on event dates
* Scary Farm fog intensity gradually increases from January 1 to opening night
* All visuals are rendered using CSS animations (no external libraries)
